# Vigenere Cipher

---
Text encryption application

> **To Encrypt**
- choose encrypt mode
- enter key
- start typing/paste plain text
- "Copy" encrypted text

> **To Decrypt**
- choose decrypt mode
- enter key
- start typing/paster encrypted text
- "Copy" decrypted text

> **"clear" to refresh or restart the cipher**
